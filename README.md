# Rohail Rahmat - Portfolio Website

A modern, responsive portfolio website showcasing System Designer & WordPress Developer expertise, built with cutting-edge web technologies.

## 🌟 Features

- **Modern Design**: Clean, professional UI with smooth animations
- **Responsive Layout**: Fully optimized for mobile, tablet, and desktop devices
- **SEO Optimized**: Meta tags, structured data (JSON-LD), and semantic HTML
- **Testimonial Carousel**: Swiper-powered carousel showcasing client reviews
- **Interactive Elements**: 
  - Animated counters
  - AOS (Animate On Scroll) effects
  - Typed.js text animation
  - Smooth scrolling navigation
- **Performance**: Fast-loading with CDN-hosted libraries
- **Accessibility**: WCAG compliant with skip links and semantic structure

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript ES6+
- **Styling**: Tailwind CSS
- **Animations**: AOS (Animate On Scroll), Typed.js
- **Carousel**: Swiper.js
- **Icons**: Font Awesome 6
- **Fonts**: Inter (Google Fonts)
- **Schema**: JSON-LD structured data

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No build process required

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/rohailrahmat/portfolio.git
cd Wordpress-Portfolio
```

2. Open in browser:
```bash
# Windows
start Wordpress-portfolio.html

# macOS
open Wordpress-portfolio.html

# Linux
xdg-open Wordpress-portfolio.html
```

Or use Live Server extension in VS Code for hot reload.

## 📋 Customization

### Change Colors
Update the blue color (#2563eb) throughout the HTML and inline styles:
```html
<!-- Header button -->
<a class="bg-blue-600">Hire me</a>

<!-- In tailwind classes -->
class="text-blue-600"
```

### Update Content
- **Hero Section**: Update name, description, and counters
- **Services**: Modify service cards and descriptions
- **Projects**: Add/remove projects in the portfolio section
- **Skills**: Update technical stack
- **Testimonials**: Add new testimonial slides
- **Contact**: Update email, phone, and social links

### Modify Animations
- AOS animations: Adjust `data-aos` and `data-aos-delay` attributes
- Typed.js strings: Update array in the `<script>` section
- Counter targets: Change `data-target` values
